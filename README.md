**LIB** - Библиотека  
**Client** - Программа №1  
**Server** - Программа №2  

# Запуск

1. Из папки проекта выполните

    ```bash
    cd src/
    ```

2. Выполните следующие команды

    ```bash
    cmake -B build
    cmake --build build
    ```

3. Запустите сервер

    ```bash
    ./build/bin/Server
    ```

4. Запустите сервер

    ```bash
    ./build/bin/Client
    ```
